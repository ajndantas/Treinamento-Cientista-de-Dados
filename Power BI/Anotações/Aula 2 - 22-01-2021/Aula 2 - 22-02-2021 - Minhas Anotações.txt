As transformações no Power Bi se refletem no Power Query (Transform Data), mas o inverso não

Ao se clicar na engrenagem, ao lado de Applied Steps, no Power Query, você os dados que você entrou, ou fórmulas.

Se eu quiser criar uma coluna, pelo Power BI, eu não preciso passar uma coluna, ele cria como nome Column, pelo Power Query, sou obrigado a passar uma

Obs: Para ver o código em M, clicar em View/Advanced Editor

Clicando no gráfico, de forma que seja a seleção total, todos os dados são utilizados. Todos os filtros são retirados.

Para não deixar marcar mais de um -> No Rolinho, Selection Control, Single Select

Para inserir Imagem -> Insert/Image

Para formatar as casas do número (centena, milhar, etc...) pode-se usar a Visão Report, clicando no campo, ou a visão data, e fazendo pela coluna

No Mapa, campo Size, permite colocar as "bolinhas" com tamanhos diferentes, conforme o campo passado nesta propriedade
